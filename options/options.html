Open Google Calendar

This extension uses the tab features of Thunderbird to open Google Voice inside a Thunderbird tab.

How it Works

This extension adds a button to Thunderbirds main tool bar as well as an item to the Tools menu which will open up Google Voice in a new tab.

On first use you should see the Google Account login page. When you login you will be prompted by a notification bar which can save your username and password for the site. This password can be removed from the Security -> Passwords section of your preferences.
